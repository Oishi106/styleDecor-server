# StyleDecor Server

A Node.js/Express server for the StyleDecor application with MongoDB integration.

## Description

This is the backend server for the StyleDecor project, providing REST APIs for managing decor items and related data.

## Installation

Install dependencies:

```bash
npm install
```

## Usage

Start the server:

```bash
npm start
```

Or with nodemon for development:

```bash
npm run dev
```

## Dependencies

- **express** - Web framework for Node.js
- **cors** - Middleware for handling Cross-Origin Resource Sharing
- **mongodb** - MongoDB driver for Node.js

## Development

- **nodemon** - Auto-restart server on file changes
